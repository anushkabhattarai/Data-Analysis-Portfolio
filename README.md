## 🔍 About Me

I'm a data and business analyst who loves the moment a spreadsheet stops looking like noise and starts telling a story. My work sits at the intersection of **business intelligence, data visualization, and analytical storytelling** — I build dashboards people actually use, clean datasets others can trust, and translate numbers into recommendations leaders can act on.

This repository is my working portfolio: a living collection of end-to-end projects spanning **Power BI, Python, SQL, and custom web & BI visuals**. Each project folder contains the raw data, the analysis, the final deliverable, and a project-specific README explaining the *so what*.
