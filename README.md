## 🔍 About Me

I'm a data and business analyst who loves the moment a spreadsheet stops looking like noise and starts telling a story. My work sits at the intersection of **business intelligence, data visualization, and analytical storytelling** — I build dashboards people actually use, clean datasets that others can trust, and translate numbers into recommendations leaders can act on.

This repository is my working portfolio: a living collection of end-to-end projects spanning Power BI, Python, SQL, and custom web dashboards. Each project folder contains the raw data, the analysis, the final deliverable, and a project-specific README explaining the "so what."

## 🎯 What I Do

- 📊 **Business Intelligence & Dashboards** — Power BI reports with custom visuals, DAX measures, and interactive drill-downs.
- 🐍 **Exploratory Data Analysis** — Python (Pandas, NumPy, Matplotlib, Seaborn) notebooks that go from raw CSV to shipped insight.
- 🗄️ **SQL & Data Modelling** — Star-schema design, relational databases, and analytical queries in PostgreSQL and MySQL.
- 🌐 **Custom Web Dashboards** — HTML / CSS / JavaScript + Chart.js dashboards for data that deserves more than a static chart.
- 🧹 **Data Cleaning & Transformation** — Reproducible pipelines that turn messy, real-world data into clean, decision-ready datasets.
